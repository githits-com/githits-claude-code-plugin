# GitHits Help

Here are the available GitHits commands:

- `/githits:search <query>` — Search GitHub repositories and code via GitHits.
- `/githits:login` — Log in to your GitHits account.
- `/githits:logout` — Log out of your GitHits account.
- `/githits:help` — Show this help message.
