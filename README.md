# GitHits Claude Code Plugin

Search GitHub directly from Claude Code using the GitHits MCP server.

## Installation
```
/plugin marketplace add githits-marketplace
/plugin install githits@githits-marketplace
```

## Commands

- `/githits:login` — Log in to your GitHits account
- `/githits:logout` — Log out of your GitHits account
- `/githits:search` — Search GitHub repositories and code

## Requirements

- Claude Code installed
- Node.js / npm (for the `@githits/mcp` package)
